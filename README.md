## 들어가며

프로그래머의 역할은? 

 요구사항을 기반으로 문제를 해결하기 위한 방안을 고안하고, 이를 코드로 구현하는 것

코드를 구현하려면 자신이 작성하는 코드의 동작을 예측할 수 있어야 함

## 학습 방법
1. **토끼굴에 빠지지 않도록** 주의해야 함 
2. 기본 개념과 동작 원리 이해했다면 이를 활용해 코드 구현 능력을 갖추기 위한 **의도적인 연습** 필요
